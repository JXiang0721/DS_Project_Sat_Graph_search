# Satellite Graph Search System (衛星圖形搜索系統)

這是一個基於 **C 語言** 開發的資料結構專題專案，旨在模擬並分析衛星網路的動態拓撲。系統透過讀取衛星軌道數據，建立鄰接鏈表（Adjacency List），並提供包含深度優先搜索（DFS）與最短路徑搜尋等多項分析功能。

---

## 🚀 系統功能

程式啟動時會隨機選擇一個當下時間（如 `2024-01-01 16:04:00`），並根據該時刻各衛星的空間位置自動建立鄰接鏈表。

### 核心模式說明：

1. **Mode 1：全節點相對距離輸出**
   計算並列出網路中所有節點對（Node Pairs）之間的相對距離。
2. **Mode 2：鄰接鏈表可視化**
   以圖形化（文字介面）呈現鄰接鏈表的結構，清楚展示各衛星節點的連接狀況。
3. **Mode 3：特定節點距離查詢**
   使用者可選定單一衛星節點，查看其與網路中其餘所有節點的精確相對距離。
4. **Mode 4：深度優先搜索 (DFS)**
   輸入起始節點，系統將執行 DFS 演算法並輸出走訪路徑，展示網路的連通性。
5. **Mode 5：最短路徑搜尋**
   輸入起始節點與目標節點，系統將計算兩者間的最短路徑與總距離。

## 🛠 檔案結構

* `code/`: 包含系統核心邏輯與演算法實作（C 語言）。
* `sat_data/`: 存放衛星軌道與位置原始數據。
* `images/`: 存放 README 使用的展示圖片。
* `Makefile`: 專案編譯自動化腳本。

---

## 💻 技術要點

* **資料結構**：使用 **Adjacency List (鄰接鏈表)** 儲存圖形結構，有效節省空間並提升搜索效率。
* **演算法**：實作 **DFS** 進行圖形走訪，以及最短路徑演算法（如 Dijkstra 概念）處理節點間的最優傳輸路徑。
* **數據處理**：解析衛星空間座標，動態計算節點間的 Euclidean Distance 並判定連通性。
